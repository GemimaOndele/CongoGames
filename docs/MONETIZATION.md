# Monetization

## Moteur gifts -> gameplay

- `Rose` : +50 points
- `Finger Heart` : +100 points
- `Lion` : x2 pendant 20 secondes
- `Galaxy` : declenche une battle
- `Universe` : declenche un bonus round

## Anti-abus

- Cooldown par joueur
- Limite par manche
- Journalisation des triggers

## UX recommandee live

- Toujours afficher "gifts = bonus gameplay"
- Afficher les events gifts dans le feed visuel
- Annoncer vocalement les gros gifts
