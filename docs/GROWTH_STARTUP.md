# Growth 10k + Startup Plan

## Optimiser pour 10k viewers

- Hook fort dans les 5 premieres secondes
- Rythme: une action visible toutes les 3-5 secondes
- Leaderboard permanent a l'ecran
- Reponses chat simplifiees (A/B/C/D)
- Debouncing backend pour absorber les pics

## KPI a suivre

- Retention 30s / 2min
- Messages chat par minute
- Gifts par minute
- Conversion viewer -> participant -> donateur

## Transformer en startup

### Etape 1: MVP
- Quiz + gifts + battles + robot host

### Etape 2: Produit
- Multi-mini-jeux + saisons + profils joueurs

### Etape 3: Business
- Sponsoring quiz, marque blanche, abonnements premium
